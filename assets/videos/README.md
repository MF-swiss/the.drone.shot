# VIDEO PLACEHOLDER

## Hero Video hinzufügen

Das Projekt erwartet ein Video unter:
`assets/videos/hero.mp4`

### Optionen:

1. **Eigenes Video hochladen:**
   - FPV Drohnen-Video (.mp4)
   - Empfohlen: 1920x1080 oder höher
   - Format: H.264 codiert
   - Länge: 10-30 Sekunden (Loop)

2. **Temporäre Lösung - Externes Video:**
   Ändere in `index.html`:
   ```html
   <!-- Statt lokaler Datei: -->
   <source src="https://example.com/dein-video.mp4" type="video/mp4">
   ```

3. **Video-Hoster verwenden:**
   - Vimeo oder YouTube embed
   - Oder Cloudflare Stream

### Video beschaffen:

- Eigene FPV Aufnahme verwenden
- Stock-Video von Pexels/Pixabay (kostenlos)
- Professionell produzieren lassen

---

**Nach dem Hochladen:** Lösche diese Datei und ersetze sie mit `hero.mp4`
